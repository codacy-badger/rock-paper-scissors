# Rock Paper Scissors
This is a Rock, Paper, Scissors multiplayer game.
<br/>
The player's name is registered after the first bet. Choose from 0 to 2 to make a bet. Press "q" to exit.
## Prerequisites
Java 8
<br/>
Maven

## Build
```
> mvn package
```
## Test
```
> mvn test
```
## Run Server
```
> java -jar target/server.jar
```
## Run Client(s)
```
> java -jar target/client.jar
```
